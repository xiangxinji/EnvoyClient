# Keyboard Shortcuts

Perform common operations quickly with global keyboard shortcuts. Shortcuts work at the system level, even when the app is not in the foreground.

## Setting Shortcuts

In the keyboard shortcuts settings panel, click the record button on the shortcut row you want to set, then press the key combination (must include a Ctrl, Alt, or Meta modifier key). Press Esc to cancel recording.

## AI Auto Reply

Press the assigned shortcut to quickly toggle AI auto reply on/off.

## AI Task Mode

Press the assigned shortcut to quickly switch task execution mode (auto/manual).

## Lock Screen

Press the assigned shortcut to immediately lock the application interface. A password is required to unlock.

## Sync Knowledge Base

Press the assigned shortcut to immediately trigger a knowledge base sync operation.

## Restore Knowledge Base

Press the assigned shortcut to restore the knowledge base from the cloud to local.
