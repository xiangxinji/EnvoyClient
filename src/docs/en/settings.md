# Settings

The settings panel provides personalization options, accessible from the avatar dropdown menu.

## Theme

Switch color themes in general settings: Default (green), Blue, or Purple. Theme changes take effect immediately.

## Language

Switch interface language in general settings: Simplified Chinese or English. Interface text updates immediately after switching.

## Auto Start

Enable or disable auto-start on boot in general settings. When enabled, the app will launch automatically when the system starts.

## Profile

In profile settings, you can modify: avatar (click to upload an image), nickname, capability tags, and responsibility description. Changes take effect after saving.

## AI Settings

In AI assistant settings, configure: auto reply toggle, AI suggestion history message count (controls context length for AI-generated suggestions).

## Task Settings

In Task & Agent settings, configure: task execution mode (auto/manual), Agent working directory.

## Lock Screen

Set a lock screen password at the bottom of profile settings. Once set, you can lock the app via keyboard shortcut or menu.
