# Chat System

The chat system supports both team channel group chat and one-on-one private messaging with rich message interactions.

## Send Messages

Type text in the input box at the bottom and press **Enter** to send. Press **Shift + Enter** to insert a line break. The rich text editor supports pasting and dragging images into the input box.

## @ Mention Members

In channel chat, type **@** to open the team member list. Select a member to mention them, and the mentioned member will be highlighted in the message.

## Quote and Reply

Hover over any message bubble to reveal action buttons. Click the reply button to quote the original message. The reply content will appear below the quote card.

## Forward Messages

Right-click a message and select "Forward". Multi-select mode is available: enter multi-select mode from the header menu, check multiple messages, and batch forward them to other members or channels.

## Revoke Messages

Right-click your own message and select "Revoke" to delete it. After revocation, other members will see a revocation notice.

## File Attachments

Click the attachment button in the input toolbar to select local image files as attachments. Attachments are displayed as thumbnails in the message.

## Stickers

Click the emoji button in the input toolbar to open the sticker panel. Select a sticker to send. Other members can right-click to save stickers to their collection.

## Cloud References

In the chat input, type a trigger to search and reference files or folders from the cloud drive. References appear as cards in the message for easy resource sharing.

## AI Suggested Replies

Click the lightning button in the input toolbar to have AI generate a reply suggestion based on the chat context. You can accept, regenerate, or dismiss the suggestion.
