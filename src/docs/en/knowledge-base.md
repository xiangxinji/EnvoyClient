# Knowledge Base

The knowledge base manages Agent skills and memories, supporting local and cloud synchronization.

## Scheduled Sync

Enable scheduled sync in knowledge base settings by setting a sync interval (hours and minutes). The system automatically syncs local brain data to the cloud.

## Post-task Sync

Enable "Sync After Task" to automatically trigger knowledge base sync after each task execution, ensuring the latest execution experience is saved.

## Reflection Memory

Enable "Reflection Memory" to have the Agent perform reflection summaries after task completion, writing lessons learned to the knowledge base to improve future task execution quality.

## Manual Sync

Click the "Sync Now" button in the knowledge base settings panel to manually trigger a sync. Progress and current file are displayed during synchronization.

## Restore Knowledge Base

In the danger zone at the bottom of the knowledge base settings panel, click "Restore" to restore the knowledge base from the cloud to local. This overwrites local data, so use with caution.
