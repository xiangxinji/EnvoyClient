# Execution Monitoring

The execution monitoring panel displays the Agent's task execution process in real-time, helping you track automation progress and details.

## Real-time Execution Log

The panel displays execution records in chronological order, including Agent reasoning, tool calls, and tool return results. New records auto-scroll to the bottom.

## Stage Grouping

Execution logs are grouped by Agent stage: Planner, Executor, Reviewer, Scorer. Each stage has completed, in-progress, and pending status indicators.

## Reasoning Process

Each reasoning step shows the Agent's thought process, helping you understand why the Agent made a particular decision.

## Tool Calls

Displays the tool names and parameters invoked by the Agent, as well as the returned results after tool execution. This helps track the Agent's specific operations.

## Execution Status

The top of the panel shows the currently executing task information and overall status. A green completion indicator appears when execution is done.
