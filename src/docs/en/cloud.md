# Cloud Resources

Shared team cloud storage for managing and storing team files.

## File Upload

Click the "Upload" button in the toolbar to select local files. Upload progress is shown with a progress bar and percentage.

## File Download

In the file list, click the download button on the right side of a file row to download it locally.

## Directory Management

Click the "New Directory" button and enter a name to create a new folder. Click a directory row to enter it, and use breadcrumb navigation to go back to parent directories.

## Batch Operations

Click the "Select" button in the toolbar to enter multi-select mode. Check multiple files or folders, then use the batch delete button to remove them at once.

## File References

In the chat input, you can reference cloud files which appear as cards in messages, making it easy for team members to quickly access shared resources.
