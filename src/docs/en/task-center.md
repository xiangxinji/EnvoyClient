# Task Center

The task center is the core panel for viewing and managing all tasks. Leaders and Members see different views.

## Current Task

In the Member view, the top shows the currently executing task card with task content and action buttons. Click "Execute Task" to manually trigger execution, or "Abort Task" to cancel an ongoing execution.

## Queued Tasks

In the Member view, queued tasks waiting for execution are displayed in order. New tasks entering the queue have entrance animations.

## Review Tasks

In the Leader view, tasks awaiting review are displayed. The Leader can view AI review results or manually click "Approve" or "Reject" buttons to make a decision.

## Status Groups

All tasks are grouped by status (completed, failed, etc.). Click a group header to expand/collapse. Status changes trigger visual feedback animations on cards.

## Task Details

Click any task card to open the task detail panel, showing full execution steps, tool call records, file resources, and review results.
