# 执行监控

执行监控面板实时展示 Agent 的任务执行过程，便于了解自动化执行的进度和细节。

## 实时执行日志

面板按时间顺序展示每一步执行记录，包括 Agent 的推理过程、工具调用和工具返回结果。新记录自动滚动到底部。

## 阶段分组

执行日志按 Agent 阶段分组展示：规划器（Planner）、执行器（Executor）、评审器（Reviewer）、评分器（Scorer）。每个阶段有完成、进行中和待执行三种状态指示。

## 推理过程

每一步推理显示 Agent 的思考内容，帮助理解 Agent 为什么做出某个决策。

## 工具调用

展示 Agent 调用的工具名称和参数，以及工具执行后的返回结果。方便追踪 Agent 的具体操作。

## 执行状态

面板顶部显示当前执行的任务信息和整体状态。执行完成后会显示绿色完成标记。
