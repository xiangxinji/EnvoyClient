# 知识库

知识库管理 Agent 的技能和记忆，支持本地与云端同步。

## 定时同步

在知识库设置中可开启定时同步，设定同步间隔（小时和分钟）。系统会自动将本地大脑数据同步到云端。

## 任务后同步

开启"任务后同步"选项后，每次任务执行完成后会自动触发知识库同步，确保最新的执行经验被保存。

## 反思记忆

开启"反思记忆"后，Agent 在任务执行完成后会进行反思总结，将经验教训写入知识库，提升后续任务执行质量。

## 手动同步

在知识库设置面板点击"立即同步"按钮，手动触发一次同步操作。同步过程中会显示进度和当前文件。

## 恢复知识库

在知识库设置面板底部的危险区域，点击"恢复"按钮可从云端恢复知识库到本地。此操作会覆盖本地数据，需谨慎使用。
