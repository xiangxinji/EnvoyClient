# Agent 执行

Agent 是自动化执行任务的智能体，通过 ReAct 循环（推理-行动-观察）完成复杂任务。

## 自动与手动模式

在设置中可切换任务执行模式：自动模式下 Agent 收到任务后立即执行；手动模式下需要手动点击"执行任务"按钮触发。

## ReAct 循环

Agent 采用 ReAct 循环执行任务：每一步先推理（分析当前状态），然后选择合适的工具执行操作，再观察结果，循环直至任务完成。最多执行 20 步。

## Shell 工具

在 Tauri 桌面环境中，Agent 可以执行 Shell 命令，用于编译代码、运行脚本等操作。执行超时 60 秒。

## 文件读写工具

Agent 可以读取和写入本地文件，用于查看代码、修改配置等操作。

## 资源上传工具

Agent 可以将本地文件上传为任务资源，方便 Leader 查看执行产出物。

## 云操作工具

Agent 可以列出团队云资源目录和上传文件到云端，实现与云盘的交互。

## 技能目录

Agent 可以读取 `~/.envoy/brains/{user}/skills/` 下的技能文件，获取特定领域的操作指南。
